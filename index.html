FAQ

Before you ask

How hot is it, really?

Scotch bonnet forward, so it arrives fast and blooms. Hotter than any grocery shelf sauce, friendlier than an extract stunt sauce. Flavor first, always.

What do I put it on?

Wings, tacos, pizza, ramen, eggs, oysters. If it is edible, it qualifies. Start with a few drops and work up.

How should I store it?

Refrigerate after opening and use within a few months. It rarely lasts that long.
